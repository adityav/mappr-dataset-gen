> Parser module dependencies to generate mappr.io dataset


## Install

```sh
$ npm install --save deps-parser
```


## Usage

```js
var deps-parser = require('deps-parser');

deps-parser('Rainbow');
```

```sh
$ npm install --global deps-parser
$ deps-parser --help
```
